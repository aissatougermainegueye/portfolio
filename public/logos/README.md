# Company Logos

Dépose ici les logos des entreprises. Formats acceptés : SVG (recommandé), PNG, JPG.

Fichiers attendus (nommage exact) :

- `technipfmc.svg` — TechnipFMC
- `covea.svg` — Covéa
- `mma.svg` — MMA

Si un fichier est absent, le portfolio affiche automatiquement les initiales à la place (ex. « TS » pour TechnipFMC Subsea France, « C » pour Covéa, « M » pour MMA).

## Où trouver les logos ?

- **Site officiel** de l'entreprise (souvent en pied de page ou dans le kit presse).
- **Wikipedia** (fichier SVG dans l'infobox).
- **Brandfetch** (https://brandfetch.com) ou **Simple Icons** (https://simpleicons.org).

## Conseils

- Préfère un **SVG** pour un rendu net à toutes les tailles.
- Idéalement un logo **carré** ou compact (le cadre affiché est un carré arrondi de ~56 px).
- Fond transparent conseillé.
